== Description ==
Settings plugin for customs configuration of Esselink.nu WP websites.

Add some extra functions to your WordPress site like auto update the Core / Themes / Plugins.

== Installation ==
Automatic Installation:

1. Go to Admin - Plugins - Add New and search for "esselink.nu"
2. Click the Install Button
3. Click 'Activate'

Manual Installation:

1. Download esselink-nu-settings.zip
2. Unzip and upload the 'esselink-nu-settings' folder to your '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
